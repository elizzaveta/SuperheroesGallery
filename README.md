## Installation

1. Clone repository

2. Run npm install for both client and server projects:
   
**Server**

```bash
$ npm install
```

**Client**

```bash
$ npm install
```


## Running the app

1. Run start command for both client and server projects:

**Server**

```bash
$ npm run start
```

**Client**

```bash
$ npm start
```

2. Open localhost:3000 
## Test

**Server**

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
